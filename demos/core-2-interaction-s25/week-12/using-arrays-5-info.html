JavaScript Arrays: Other Kinds of Data

Arrays can story any kind of data. In this demo, our array is storing URLs to photos of tuxedo cats. We then target an <img> element. When we run our function, we want to change the image’s src HTML attribute. This is actually really easy! Here’s how:

  1. Target the element and assign it to a variable. For example, let’s say our variable is called output.
  2. To access any HTML attribute, write the variable name followed by the attribute name. For example, if we’re trying to get the src attribute, we write output.src.
  3. Then set that attribute equal to a new value. For example: output.src = "new value".

Other common attribute examples: .id, .href, .value (for input elements).
