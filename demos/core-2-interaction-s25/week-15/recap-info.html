Recap So, What Did We Learn?

Since we started learning JavaScript, we covered the following topics:

Week 11

  • Adding JavaScript using the <script> element
  • Defining variables using let
  • Basic types (numbers, strings, Booleans)
  • Outputting data to the console using console.log()
  • Checking data types using typeof()
  • Arithmetic, string, comparison, and logical operators
  • Functions, including function arguments/parameters and using return to return values
  • Conditionals (i.e. if else statements)
  • Targeting elements by ID or class using document.querySelector()
  • Making elements interactive using .addEventListener()
  • Creating popup dialogs with alert()
  • Modifying inline CSS using .style.__propertyName__
  • Adding, removing, or toggling CSS classes using .classList

Week 12

  • Common bugs from Week 11
  • Changing content inside and element using .textContent and .innerHTML
  • Arrays and array indices
  • Handling edge cases in arrays and using .length
  • Global vs. local scope
  • Modifying element values like src and href
  • Generating a random number using Math.random()
  • Converting that number to a specific range
  • Also, rounding a number using Math.floor(), Math.ceil(), or Math.round()
  • Applying randomization to CSS properties
  • Getting random entrys from an array

Week 13

  • Creating new elements using document.createElement()
  • Adding elements inside of other elements using .append()
  • Creating complex strings using template literals
  • Repeating code using setInterval()
  • while loops, as well as loop counters, loop conditions, and loop increments
  • Avoiding infinite loops
  • for loops
  • for...of loops for looping through an array
  • Targeting an array of elements using document.querySelectorAll()
  • JavaScript libraries, like Tone.js

Week 14

  • Getting data from <input> elements
  • Using that data to update CSS
  • JavaScript objects
  • Using an object to track settings
  • The Document Object Model (DOM)
  • APIs
