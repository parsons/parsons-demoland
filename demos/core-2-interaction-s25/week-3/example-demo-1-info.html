CSS An article card component

Here is some sample HTML for a simple article component with a title, brief text, an image, and a link to read more.

To get setup, you will need to create a stylesheet titled main.css in the same folder as your index.html document, and copy and paste the styles from the bottom of this document.

We will also be using for the first time a reset stylesheet, which helps to clear out the user agent styles to give you a clean starting point. You will need to create a stylesheet titled reset.css in the same folder as your index.html document and paste these CSS rules into it.
