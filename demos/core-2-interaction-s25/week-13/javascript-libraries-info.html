JavaScript Libraries

By now, you’ve learned about most of the fundamental concepts of JavaScript (and actually of all programming languages)! What’s really cool about JavaScript is that we can use these simple fundamentals to make incredibly complex things.

When people use JavaScript to accomplish complex tasks, they often release their code for others to use as well. To do so, they create JavaScript libraries, which are essentially just massive .js files that includes unique functions.

This demo uses the Tone.js JavaScript library, which adds audio synthesis functionality to websites. Every library features its own documentation so that people can learn how to use it. Documentation can often be pretty hard to read, but luckily we don’t need to understand ALL of it to make use of a library. This demo features a fairly simple example of Tone.js used to add dynamic audio.

NOTE: to play audio in a web browser, the user needs to click somewhere on the page first! This is a security limitation of web browsers.
