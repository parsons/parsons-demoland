Recap Weekly Challenge: Digital Instrument

For this week’s challenge, you’ll be creating a website that acts like a kind of digital musical instrument, or a piece of generative art. I’ve already started it for you, so here’s what you need to finish:

  1. Make the “for” loop run a random number of times, between 0 and 100
  2. Create a <div> element inside of the loop
  3. Finish the first template literal with the randomized red, green, and blue values
  4. Add the new element to the container
  5. Finish the second template literal with the randomized red, green, and blue values
  6. Use setInterval to run this function every 1000ms

I’ve added TODOs to the code to indicate where you should write your own code. Make sure to remove the full /* TODO */ text, otherwise you’ll be left with comments that ruin your code!

If you’re looking for an optional challenge, here’s what else you can try:

  • Try making the generated div elements use pre-specified colors instead of totally random colors
  • Try editing more CSS properties of the generated div elements
  • Try editing the Tone.js code to play specific notes instead of random notes
  • Try using document.querySelectorAll() to change something about ALL generated divs, even after they’ve already been generated
