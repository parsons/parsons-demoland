JavaScript Functions

Functions let us run an chunk of code all at once. We can reuse functions with different inputs, which we call either arguments or parameters.

We define a function by doing the following:

  1. Start by writing function followed by the function’s name. This can be whatever you want as long as it’s one word without punctuation.
  2. In parentheses, define one or more arguments separated by commas. These are like variables that are only available inside of your function.
  3. In curly brackets, write as much code as you want! And then if you want your function to output something, you can use the return keyword followed by a value.

We can call (i.e. run) a function by writing its name and feeding it some values.
