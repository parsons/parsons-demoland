JavaScript Modifying CSS

Now let’s move on to some fun *visual* stuff.

In this demo, we’re using JavaScript to change the CSS styles of our element. Here’s how we do that:

  1. First, we need to assign our element to a variable like before.
  2. Then, we set .style.__propertyName__ equal to something to change its value.

This works for any CSS property. BUT, because we can’t use hyphens (-) in JavaScript since they mean subtraction, we have to use camelCase notation instead. So, a property like background-color would become backgroundColor.
