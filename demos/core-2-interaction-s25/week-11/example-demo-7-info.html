In this demo, I’ve added some JavaScript to our HTML document by using the <script> element. JavaScript has its own unique syntax different from HTML and CSS. The first thing to note is that in JavaScript, we usually end every line with a semicolon ;

Variables and data types

This demo features several variables. Variables let us keep track of data and refer to it later on. There are several types of data we’re able to store. Here are the basics:

  • Numbers — either integers (rounded numbers) or floats (with decimals)
  • Strings — text, written in single or double quotes
  • Booleans — either true or false, written in lowercase without quotes

We can define variables by using the let keyword and giving our variable a name. Names must be one word without spaces. Once we’ve defined a variable, we can redefine it at any time.

Testing JavaScript

Open the console by pressing the bottom-left button in the Editor panel. The console is your interface for testing JavaScript code. (BTW, this is a fake console I made just for the DEMOLAND site. There is also a real in-browser console, which you can access via the Inspect panel in your browser).

Inside the console, try typing console.log(var1). We can use console.log() to view the values of any variable we’ve already defined.

Now try typing console.log(typeof(var1)). We can use typeof() to view the type of any value.
