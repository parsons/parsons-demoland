JavaScript A JSON Object

JSON stands for JavaScript Object Notation. It's a way of organizing data using key-value pairs wrapped in curly braces {}.

Structure

A JSON object looks like this: 

{
  "key": "value",
  "anotherKey": 42
}

Each key is a string (in quotes), followed by a colon, then a value. Values can be strings, numbers, booleans, arrays, or even other objects.

Accessing data

We use dot notation to access values by their key. For example, person.name gives us "Ada Lovelace".

You can also use bracket notation: person["name"] does the same thing.

Try it

Open the console and try console.log(person) to see the full object. Then try accessing individual properties like console.log(person.skills).
