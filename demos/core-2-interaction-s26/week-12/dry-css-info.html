CSS DRY in CSS

DRY stands for Don't Repeat Yourself. It's a principle that says: if you're writing the same code in multiple places, there's probably a better way.

The opposite is sometimes called WET — "We Enjoy Typing." In this example, .card-1, .card-2, and .card-3 all have the exact same styles. If you wanted to change the color, you'd have to change it in three places.

The fix

Instead of giving each card its own class with duplicate styles, give them all a shared class like .card. Now the styles live in one place — change it once, and it updates everywhere.

Try it: uncomment the .card rule in the CSS, then change the class names in the HTML from card-1, card-2, card-3 to just card. Then delete the three repetitive rules.
