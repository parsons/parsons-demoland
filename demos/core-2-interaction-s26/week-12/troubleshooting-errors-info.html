JavaScript Troubleshooting Errors

Errors are a normal part of writing code. The browser's console is the first place to look when something isn't working. Right-click the page, choose "Inspect," and click the "Console" tab.

Common error types

SyntaxError — you made a typo in the code's structure. Missing quotes, parentheses, or brackets. The browser can't even run the code.

ReferenceError — you're using a variable that doesn't exist. Usually a spelling mistake.

TypeError — you're trying to do something with a value that doesn't support it, like calling .textContent on something that's undefined.

Tip

When a SyntaxError stops the script, nothing below it runs either. Fix errors from top to bottom — the first one might be causing the others.
