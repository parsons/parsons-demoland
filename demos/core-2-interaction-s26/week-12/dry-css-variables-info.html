CSS DRY CSS with Variables

What if you want to change your brand color? Right now, coral appears in three different rules. If you want to switch to steelblue, you'd have to find and update each one.

CSS custom properties (also called CSS variables) let you define a value once and reuse it everywhere. You declare them with a double dash: --brand-color: coral; and use them with var(--brand-color).

Try it

Uncomment the DRY version in the CSS and delete the WET rules above it. Then try changing the value of --brand-color to something else — the entire page updates in one edit.

Where to declare variables

Putting variables on :root makes them available everywhere on the page. You can also put them on a specific element to limit their scope.
