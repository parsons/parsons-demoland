JavaScript A Calculator with Functions

Now we have four small functions — add, subtract, multiply, and divide — and one bigger function, calc, that decides which one to call.

Functions calling functions

calc takes three arguments: an operation (a string like "add") and two numbers. It uses if statements to figure out which math function to run, then returns the result. 

calc("add", 10, 5)      // 15
calc("subtract", 10, 5) // 5
calc("multiply", 10, 5) // 50
calc("divide", 10, 5)   // 2

Why this matters

Breaking code into small, focused functions keeps things organized. Each function does one thing. The calc function doesn't need to know how to add — it just calls add() and lets that function handle it.

Try it

Try adding a new operation — like modulo (the % operator, which gives you the remainder). Write a modulo() function and add another if block in calc.
