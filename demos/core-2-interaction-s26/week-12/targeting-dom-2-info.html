JavaScript Changing Content with JavaScript

In the previous demo we changed an element's style. Now we're changing what's inside the element — its content.

textContent

element.textContent = "new text" replaces everything inside an element with plain text. Any HTML tags inside will be removed.

innerHTML

element.innerHTML = "new <strong>HTML</strong>" lets you insert HTML tags that the browser will render. Use this when you want bold, italic, links, or other elements inside.

Try it

Try changing the text and HTML in the function calls. What happens if you use changeText on the bio instead of changeHTML? The <strong> and <em> tags will show up as plain text instead of being rendered.
