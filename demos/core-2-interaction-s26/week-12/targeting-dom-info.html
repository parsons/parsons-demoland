JavaScript Targeting DOM Elements

Before JavaScript can change anything on a page, it needs to find the element first. document.querySelector() lets you target any element using the same selectors you already know from CSS.

querySelector

document.querySelector("#first") finds the element with the id "first." document.querySelector(".special") finds the first element with the class "special." The selectors work just like CSS — # for ids, . for classes.

Wrapping it in a function

Instead of writing document.querySelector() and setting .style.backgroundColor every time, we wrote a changeBackground() function that takes two parameters: a selector and a color. Now recoloring any element is a single line.
