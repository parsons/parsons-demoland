JavaScript Intro to Functions

A function is a reusable block of code. You give it a name, tell it what inputs (called parameters) it expects, and it gives you back a result with return.

Structure

function add(a, b) {
  return a + b;
}

a and b are parameters — placeholders for whatever values you pass in. When you call add(2, 3), those values are called arguments.

Why use functions?

Without a function, you'd write 2 + 3, then 10 + 20, then 100 + 1 — repeating the same logic. A function lets you write it once and call it as many times as you want.
