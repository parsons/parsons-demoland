CSS Flexbox display: wrap and direction

"flex-wrap" is effective when we have a flexbox layout. It applies to the container level, with options of "nowrap", "wrap", and "wrap-reverse".

"flex-direction" defines the main axis of the flex box — the direction in which the items are stacked. We also have a couple options to use: "row (default)", "row-reverse", "column", and "column-reverse".
