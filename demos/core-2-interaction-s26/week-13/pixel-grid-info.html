Loops Pixel Grid

A nested loop — a loop inside a loop — gives you two dimensions. The outer loop counts rows (y), the inner loop counts columns (x). Together they produce a 10×10 grid of 100 cells.

Each cell gets a random color using Math.random(). The loop provides the structure, randomness provides the variation.

Try it

Change the grid from 10 to 20 (update both the loop limits and the repeat() value). Try replacing the random color with something based on x and y — like rgb(x * 25, y * 25, 150) — to see a gradient instead of noise.
