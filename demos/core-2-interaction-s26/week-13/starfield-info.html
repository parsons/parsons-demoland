Loops Starfield

A single loop places 50 white circles on a black background. Each circle gets a random position and a random size. The result looks like a night sky — but it's the same for loop as the other two examples.

This is the key idea: the loop is always the same machine. The creative decisions happen inside the loop body. Stripes, grids, and scattered fields all come from the same for statement — the difference is what you do with each iteration.

Try it

Change 50 to 500 for a denser sky. Replace the white with a random color. Try making the opacity random too — div.style.opacity = Math.random() — for depth.
