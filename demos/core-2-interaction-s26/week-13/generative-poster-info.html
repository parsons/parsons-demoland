Exercise Generative Poster

A nested loop fills a grid with colored cells. Some cells randomly get a word in a random color — so the text is part of the grid, not floating on top of it. Every reload produces a different poster.

Your task

Tweak the parameters at the top of the script to make a poster you're proud of. When you have a result you like, screenshot it.

Things to try

  • Swap the colors array for your own palette
  • Change the words array to your own set of words
  • Raise or lower textChance to control how many cells get text
  • Change columns and rows to resize the grid
  • Change cellSize to make the pixels bigger or smaller
  • Use randomInt() to vary the font size range
  • Break the system somehow

When you're happy with your poster, screenshot it.

Stretch goal

Figure out how to turn the following into parameters:

  • Min text size
  • Max text size
  • Border radius
  • Separate cell width and cell height
