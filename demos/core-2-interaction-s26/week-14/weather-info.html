Control Flow Weather

Same if / else structure as the previous demo, but now the data is real. The page fetches the current temperature in New York City from the Open-Meteo API and runs it through the same branching logic.

fetch() makes a request to an external server. The .then() chain waits for the response, converts it to JSON, and pulls out data.current.temperature_2m. From there it's the same if / else you already know.

Try it

Change the latitude and longitude in the URL to get the weather for a different city. Try adding a background color that changes with the temperature too.
