Timing Clock

setInterval runs a function on a repeating timer. Here it fires every 1000 milliseconds — once per second — reads the current time, and updates the display. No user action required. The DOM is in continuous motion.

This is a different kind of repetition than a loop. A for loop runs immediately and finishes. setInterval runs forever, on a schedule, independent of anything the user does.

Try it

Change 1000 to 100 and add milliseconds to the display. Try changing the background color every second based on the current seconds value.
