Events Five Events

One page, five different event listeners. Each one listens for a different kind of user action and reacts to it. The browser is constantly firing events — your code decides which ones to care about.

load — runs once when the page opens. Sets a greeting and starting color based on time of day.
click — picks a random hue each time you click.
mousemove — your cursor's horizontal position controls color saturation. Move left for gray, right for vivid.
scroll — the heading grows as you scroll down the page.
keydown — press any key and it appears in the corner.

Try it

Try changing what each event does. What if scroll changed the hue instead? What if keydown changed the background to a specific color for each letter?
