Control Flow If / Else

This page checks the current hour using new Date().getHours() and picks a background color based on the time of day. Morning is gray, afternoon is blue, evening is black.

The program makes a decision using if, else if, and else. Only one branch runs — whichever condition is true first. This is how programs branch.

Try it

Change the hour thresholds or add more branches. What if you had a different color for every 3-hour window? Try setting var hour = 8 to test a specific time without waiting.
