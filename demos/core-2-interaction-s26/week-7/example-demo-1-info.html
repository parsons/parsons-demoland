CSS Animation

The CSS animation property is actually a big shorthand. In order, it takes values for:

  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
  • animation-fill-mode

Check out the MDN page for CSS animation to get to know what each of the properties controls, and know the available values for all of them.
