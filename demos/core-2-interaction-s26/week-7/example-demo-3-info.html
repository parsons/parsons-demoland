CSS Grid display: sample layout

CSS grid is a display value, it controls how the element participates in the document flow like `block` or `inline` values. Like `table`, grid layout enables an author to align elements into columns and rows. However, many more layouts are either possible or easier with CSS grid than they were with tables. The Grid Layout Module allows developers to easily create complex web layouts, and it makes it easy to design a responsive layout structure.

Let’s try it out with properties like `grid-template-columns`, `grid-column`, and `grid-row`.
