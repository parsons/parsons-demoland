JavaScript Iterate through data in an array

As a programming language, JavaScript stays true to the fundamental purpose of computers: counting and tabulating. Let's take a look at how JavaScript's mathematical basis helps us access data and work with dynamic content.

array is a data type in JavaScript, along with string, number, boolean, object, undefined and so on. There are a couple useful methods available when get to work with arrays:

  • arrayName[index] index is the element's position in the array. For instance, in an array named "alphabets" [a, b, c], b is at the second place and has an index number of 1, hence alphabets[1] would return "b" as result.
  • arrayName.length tells us how many items are in the array. The array const alphabets = [a, b, c] has 3 items, hence alphabets.length would return "3".
